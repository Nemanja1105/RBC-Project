# RBC-Project
## Technologies Used:
  - **Angular 18**
  - **Spring boot**
  - **MySQL**
  - **NginX**
## Requirements:
  - **Docker engine**
## Installation
1. Clone the repository
   ```sh
   https://github.com/Nemanja1105/RBC-Project.git
   ```
2. Go to cloned directory
3. Start the project containers using Docker Compose
   ```sh
   docker compose up -d
   ```
4. When the containers are launched, go to the address
   ```sh
   http://localhost
   ```
